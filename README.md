# Medical Analysis Dashboard 🏥

A responsive web application for analyzing medical data and presenting results in a clear, user-friendly interface. Focuses on efficient data handling and modern web technologies.

## Features
- Upload and analyze medical datasets
- Interactive dashboard with charts and statistics
- Data filtering and sorting
- Responsive UI for desktop and mobile
- Clean visualization with Tailwind CSS and Shadcn UI

## Tech Stack
- **Frontend:** React, Tailwind CSS, Shadcn UI
- **Charts:** Recharts / Chart.js
- **Icons:** Lucide React
